API Endpoints (13 rotas)
/api/auth/[...nextauth]API
NextAuth Handler
GET/POST - Gerencia sessões, login/logout e callbacks do NextAuth v5.
/api/auth/registerAPI
Registro
POST - Cria usuário com hash bcrypt, status PENDING. Valida email único.
/api/auth/forgot-passwordAPI
Esqueceu a Senha
POST - Gera token de verificação (1h), envia email via Resend. Sempre retorna sucesso.
/api/auth/reset-passwordAPI
Redefinir Senha
POST - Valida token, hash nova senha, atualiza usuário, deleta token usado.
/api/auth/complete-profileAPI
Completar Perfil
POST - Atualiza perfil com role, skills, bio. Cria skills inexistentes. Requer autenticação.
/api/uploadAPI
Upload de Arquivo
POST - Upload via Vercel Blob, retorna URL. Requer autenticação.
/api/mentorsAPI
Buscar Mentores
GET - Lista mentores APPROVED do tenant. Filtro por skill, busca por nome. Inclui contagem de conexões.
/api/connectionsAPI
Conexões
GET/POST/PATCH - Lista conexões, cria solicitação (com waitlist se lotado), aceita/rejeita (com promoção de fila).
/api/notificationsAPI
Notificações
GET/PATCH - Lista notificações do usuário. Marca individual ou todas como lidas.
/api/skillsAPI
Habilidades
GET/POST - Lista skills (global + tenant). Cria novas com verificação de duplicata.
/api/libraryAPI
Biblioteca
GET/POST - Lista materiais do tenant. Cria item (requer ADMIN ou MENTOR).
/api/admin/usersAPI
Admin: Usuários
GET/PATCH - Lista com filtros. Aprova/rejeita/suspende. Envia email e notificação ao aprovar.
/api/admin/reportsAPI
Admin: Relatórios
GET - Stats do tenant, conexões por mês, waitlist, library items, top 10 skills por uso.
